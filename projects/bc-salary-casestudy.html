Background

I'm a Vancouver-based data analyst with a Master of Business Analytics from UBC and a deep focus on British Columbia's public sector data ecosystem. My work sits at the intersection of technical rigour and clear communication — building dashboards and analytical tools that don't just show numbers, but make them matter to the people who rely on them.

I'm drawn to public sector data — housing, transit, health, education, compensation — because the stakes are real. These are systems that affect hundreds of thousands of British Columbians, and the data that describes them is publicly available but rarely made accessible. My portfolio is an attempt to change that, one project at a time.

"The data exists. The question is whether anyone has made it legible."

My technical toolkit spans the full analytics stack: Python and pandas for data extraction and pipeline engineering, Power BI and DAX for semantic model development, D3.js for custom interactive visualisation, and SQL for data wrangling. I've built automated PDF extraction pipelines using pdfplumber, vendor enrichment systems using the Claude API and BC Registry data, and GitHub Actions pipelines for scheduled data refreshes.

Currently pursuing PMP and ECBA certifications to complement the technical side with structured project management and business analysis frameworks — rounding out a profile suited to BC public sector and research analyst roles.
